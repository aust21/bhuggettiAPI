API Documentation

This document provides information about the available API endpoints for the bhuggeti API.

GET /api/questions

Retrieves a specified number of random questions from the database.

Query Parameters

  • count (optional): The number of random questions to retrieve. Defaults to 1 if not specified. If the number is out of range, available questions will be retrieved.
  • type (optional): Gets either culture fit or technical questions. Defaults to technical questions if not specified. Returns a response of invalid if the type is invalid.
  • field (optional): Gets the field of the question. Defaults to "not relevant" if not specified

Response

Returns a JSON object containing an array of question objects.

        {
            "data": [
                {
                    "question": "What is your greatest strength?",
                    "domain": "Software Engineering",
                }
            ]
        }

Example Request with count

GET /api/questions?count=2

Example Response

          {
            "data": [
                {
                    "question": "Tell me about a time you demonstrated leadership.",
                    "domain": "Software Engineering",
                },
                {
                    "question": "How do you prioritize your tasks?",
                    "domain": "Business Management",
                }
            ]
          }

Example Request with type

GET /api/questions?type=culture

Example Response

        {
          "data": [
              {
                  "domain": "Not Relevant",
                  "question": "What are your values?",
              }
          ]
        }

Example Request with count and type

GET /api/questions?count=2&type=culture

Example Response

        {
          "data": [
              {
                  "domain": "Not Relevant",
                  "question": "What do you think about global warming?",
              },
              {
                  "domain": "Not Relevant",
                  "question": "What does global warming think about you?",
              }
          ]
        }

Example Request with count and type and field

GET /api/questions?count=2&type=culture&field=architecture

Example Response

        {
          "data": [
              {
                  "domain": "architecture",
                  "question": "What is a house?",
              },
              {
                  "domain": "architecture",
                  "question": "A house is what?",
              }
          ]
        }
